USER MANUAL

1. Register API Client/User who intend to use and endpoint

 (a). API key will be generated for the registered user and is accessed in inactive key pane

2. Activate registered client as a user

3. Activate client API key

 (a). Upon key activation, client will receive an email notifyinf them of their key activation

4. Assign registered end points to the user

NB.

 - Only active users / clients with active API Keys will be assigned end points

 - An endpoint can be removed from a user

 - API Keys and users can as well be deactivated upon need to block user access to any endpoint
